Gestão de Receitas e Estrutura • Organização Hierárquica: Podes organizar receitas por Secções Principais (ex: FiveM, Oficina, Casa) e Subcategorias. • Criação Detalhada: Cada receita guarda o nome, categoria e uma lista de ingredientes com quantidades e unidades específicas. • Favoritos: Podes marcar as tuas receitas preferidas para acesso rápido. • Pesquisa Inteligente: Filtro em tempo real para encontrar receitas rapidamente.

Controlo de Stock (Inventário) • Gestão de Materiais: Existe um inventário onde podes controlar a quantidade de cada material que tens guardado. • Produção Inteligente: Ao clicar em "Produzir" numa receita, a app verifica se tens materiais suficientes e abate-os automaticamente do stock. • Feedback Visual: A app avisa-te (em vermelho) o que falta e quanto falta para conseguires completar uma receita. • Flexibilidade: Podes desativar o stock em secções onde ele não seja necessário.

Persistência e Segurança de Dados • Importação/Exportação (JSON): Podes fazer backups manuais de tudo (receitas, stock e categorias) para um ficheiro e partilhá-lo ou restaurá-lo mais tarde. • Proteção ProGuard: A app está configurada para que, mesmo na versão final (Release), os teus dados não se corrompam. • Limpeza Total: Opção para resetar a oficina e apagar todos os dados se quiseres começar do zero.

Sistema de Atualizações e Build • Ligação ao GitHub: A app agora "fala" com o teu repositório e avisa o utilizador se houver uma versão nova para descarregar. • Ambiente Dual (Debug/Release): Podes ter a versão de desenvolvimento e a versão final instaladas ao mesmo tempo no telemóvel (a de desenvolvimento tem um sufixo ".debug").

Interface (UX/UI) • Design Responsivo: Os diálogos adaptam-se para não cortarem texto em ecrãs mais pequenos. • Menu de Definições: Centralizado na "roda dentada", com uma secção "Acerca da Aplicação" onde se gere a versão e as atualizações.
